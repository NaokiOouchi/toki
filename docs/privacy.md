# プライバシーポリシー

**最終更新日：2026 年 5 月 24 日**
**適用バージョン：Toki v1.0 以降**

Toki（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、データの取り扱いに
細心の注意を払います。本ポリシーは、本アプリが取得・処理・保存するデータと、
その目的について説明します。

---

## 1. 本アプリについて

Toki は macOS 専用の常時前面表示型カレンダー時計アプリです。Google Calendar と
連携し、ユーザーの予定を円形時計上に表示します。

---

## 2. 取得するデータ

### 2.1 Google Calendar データ

ユーザーが Google アカウントでサインインした場合、Google Calendar API を通じて
以下のデータを取得します：

- 予定のタイトル
- 予定の開始時刻 / 終了時刻
- 予定の場所
- 予定の参加者情報
- 予定の説明
- 予定の URL（Google Meet / カレンダー詳細）

### 2.2 認証情報

- Google OAuth による access token / refresh token

### 2.3 アプリ設定情報

- テーマカラー、フォントサイズ、透過率などの見た目設定
- 表示する時刻範囲などの動作設定

### 2.4 クラッシュレポート / 診断情報

Apple の MetricKit フレームワークを通じて、以下が Apple サーバーに送信される
場合があります：

- クラッシュ時の診断情報
- アプリのパフォーマンス情報

これらは Apple が匿名化したうえで開発者に提供されるもので、個人を識別する情報は
含まれません。

---

## 3. データの利用目的

| データ | 利用目的 |
|---|---|
| Google Calendar 予定 | 円形時計への描画 / 詳細表示 / 外部リンク（Meet / カレンダー）への遷移 |
| OAuth トークン | Google Calendar API の認証 |
| アプリ設定 | ユーザー指定の見た目 / 動作の維持 |
| クラッシュレポート | アプリ品質の改善 |

---

## 4. データの保存場所

| データ | 保存場所 | 永続性 |
|---|---|---|
| Google Calendar 予定 | Mac のメモリのみ | アプリ終了で消失 |
| OAuth トークン | macOS Keychain | アプリ削除 / サインアウトまで保持 |
| アプリ設定 | macOS UserDefaults | アプリ削除まで保持 |
| クラッシュレポート | Apple サーバー（一時保管後 Apple ポリシーに従い削除）| – |

---

## 5. 第三者への提供

本アプリは、ユーザーのデータを Apple および Google（API 提供元）以外の第三者に
**一切提供しません**。

- 広告ネットワーク：使用していません
- 分析ツール（Google Analytics 等）：使用していません
- 独自サーバー：保有していません

---

## 6. データの削除方法

### 6.1 ローカルデータの削除

- macOS のアプリケーションフォルダから Toki を削除すると、UserDefaults と
  Keychain 内のデータも削除されます

### 6.2 Google 側の連携解除

- [Google アカウントのアプリ連携設定](https://myaccount.google.com/permissions)
  から Toki のアクセス権を削除できます

### 6.3 クラッシュレポートの送信停止

- macOS のシステム設定 > プライバシーとセキュリティ > 解析と改善 >
  「Mac の解析を共有」を OFF にすることで、Apple へのクラッシュレポート送信を
  停止できます
